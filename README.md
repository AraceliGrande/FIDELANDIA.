## Descripción del Proyecto
Fidelandia es un sistema de información desarrollado para optimizar la gestión de producción, ventas y control de stock en una empresa de elaboración de pastas. Actualmente la empresa maneja procesos manuales, lo que dificulta la trazabilidad y el control de inventario y cuentas con proveedores.  

El sistema permite consolidar información de manera confiable y accesible, apoyando la toma de decisiones y mejorando los procesos operativos y administrativos.  

---

## Funcionalidades Principales
- **Gestión de Proveedores:** Alta, actualización y consulta de proveedores; registro de transacciones y control de cuentas corrientes.  
- **Producción:** Registro de tipos de pasta, ingreso de lotes producidos y consolidación diaria de la producción.  
- **Ventas:** Registro de ventas diarias vinculadas a lotes de producción, asegurando consistencia del stock.  
- **Stock:** Visualización de stock consolidado, generación de reportes e indicadores sobre producción y ventas.  
- **Reportes y dashboards:** Visualización de balances diarios, gráficos por tipo de pasta y exportación de datos para análisis.

##  Instalación del Proyecto

1. **Clonar repositorio:**
```bash
git clone <URL-del-repositorio>
```
2. **Abrir solución en Visual Studio 2022 o superior:**
- Seleccionar la carpeta del proyecto .sln
- Restaurar paquetes NuGet si es necesario
3. **Configurar Base de Datos**
- Crear base de datos en SQL Server
4. **Ejecutar la aplicación**
- Presionar F5 o Run en Visual Studio
